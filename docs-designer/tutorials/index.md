# Tutorial Library

Sixteen follow-on tutorials that build real skins after the lead
[Blue Morpho butterfly tutorial](../getting-started/butterfly/index.md).
Sorted by difficulty.

## Beginner

| Tutorial | Time | What you make |
|---|---|---|
| [Borderless music widget](02-borderless-music-widget.md) | 25 min | A star-shaped music widget skin |
| [Star-shaped app window](03-star-shaped-app-window.md) | 20 min | Practice authoring SVG window shapes |
| [Custom brush from a photo](04-custom-brush-from-photo.md) | 25 min | Capture a tileable from a photo, save as a brush |
| [Import a Figma screen](05-import-a-figma-screen.md) | 20 min | Bring a Figma frame into the Designer |
| [Lottie loading spinner](06-lottie-loading-spinner.md) | 25 min | Author a loading spinner from a Lottie file |

## How to read them

Each tutorial:

- Hero shot at the top of the finished result.
- Prerequisites (which earlier tutorial / which Designer feature
  is required).
- Step-by-step, screenshot-light, with exact menu paths.
- Closes with "what you exercised" and see-also links.
