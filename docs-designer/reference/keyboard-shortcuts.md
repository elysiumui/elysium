# Keyboard shortcuts

Comprehensive Designer keymap. Bindings shown for macOS first, then
Windows / Linux. Where they differ, both are listed.

## Global

| Shortcut | Action |
|---|---|
| `Cmd+N` / `Ctrl+N` | New skin |
| `Cmd+O` / `Ctrl+O` | Open skin |
| `Cmd+S` / `Ctrl+S` | Save |
| `Cmd+Shift+S` / `Ctrl+Shift+S` | Save As |
| `Cmd+W` / `Ctrl+W` | Close window |
| `Cmd+Q` / `Ctrl+Q` | Quit Designer |
| `Cmd+Z` / `Ctrl+Z` | Undo |
| `Cmd+Shift+Z` / `Ctrl+Shift+Z` | Redo |
| `Cmd+,` / `Ctrl+,` | Preferences |
| `Cmd+/` / `Ctrl+/` | Open Aether chat |

## Brush

| Shortcut | Action |
|---|---|
| `[` / `]` | Smaller / larger brush |
| `Shift+[` / `Shift+]` | Lower / higher opacity |
| `Ctrl+[` / `Ctrl+]` | Lower / higher hardness |
| `B` (hold) | Quick Wheel |
| `Alt` + click | Sample color |
| Left-click on palette slot | Open the **Color menu** (Pick color / Apply / Clear / Cancel) |
| Right-click on palette slot | Open the **Texture menu** (tile grid / Import / Generate / Apply / Clear / Cancel) |
| Esc | Dismiss the open palette menu without changes |

## Animation

| Shortcut | Action |
|---|---|
| `Space` | Play / pause |
| `Home` / `End` | Go to start / end |
| `Left` / `Right` | Step back / forward frame |
| `,` / `.` | Step back / forward key |
| `S` | Set Key |
| `Shift+S` | Set Breakdown |
| `G` | Open Graph Editor |

## Aether

| Shortcut | Action |
|---|---|
| `Cmd+/` / `Ctrl+/` | Open chat |
| `Cmd+K` / `Ctrl+K` | Quick command (within chat) |
| `Esc` | Close chat panel |

## Customizing

`File > Preferences > Keyboard Shortcuts` opens the customizer.
Click any binding to record a new one. Reset individually or as a
group.

## Printable cheat sheet

A PDF cheat sheet ships with the Designer at
`Help > Print Shortcut Cheat Sheet`. Or download from
[elysiumui.com/cheatsheet.pdf](https://elysiumui.com/cheatsheet.pdf).
